# world_time

Flutter project.

This app get time of defferent reigions using world time API (http://worldtimeapi.org/).
User inputs timezone using a list view.

## dependencies
  http: ^0.12.0+2
  intl: ^0.15.8
  flutter_spinkit: ^4.0.0

Images are contained in assets directory.
